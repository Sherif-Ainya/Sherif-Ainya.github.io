Automated Robotic Arm System

Completed: June 2022
Categories: Robotics, Automation, Control Systems
Team Size: 4 members

Project Overview

This project involved designing and building an automated robotic arm capable of precise object manipulation in a manufacturing environment. The system integrates mechanical components, electronic sensors, and control software to achieve high accuracy and repeatability.

Key Features

  • 6-axis movement with ±0.1mm precision
  • Computer vision for object recognition
  • PLC-based control system
  • Custom-designed gripper mechanism
  • Real-time monitoring interface

Technical Specifications

Mechanical

  • Reach: 800mm
  • Payload: 2kg
  • Repeatability: ±0.1mm
  • Actuators: Stepper motors

Electrical

  • Power: 24V DC
  • Control: Arduino + PLC
  • Sensors: Encoders, Force sensors
  • Vision: Raspberry Pi camera

Software

  • Control: C++
  • Vision: Python/OpenCV
  • Interface: Custom GUI
  • Communication: Modbus TCP

Project Gallery

Challenges & Solutions

Achieving the required precision was challenging due to mechanical backlash. We implemented software compensation algorithms and used higher quality harmonic drives to minimize positional errors.

Initial vision system had trouble with reflective surfaces. We added diffuse lighting and implemented edge detection algorithms that were less sensitive to reflections.

Integrating mechanical, electrical and software components required careful timing and synchronization. We developed a state machine architecture that clearly defined system states and transitions.
